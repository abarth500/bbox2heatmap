# bbox2heatmap

## Install

```(JavaScript)
npm install bbox2heatmap
```

Set your Flickr API key and secret into value.js (The themplate is value-original.js)

```(JavaScript)
{
  "flickr_options": {
    "api_key": "01234567890123456789012345678901",
    "secret": "1234567890123456"
  }
}
```

## Test

```(JavaScript)
node test/test.js
```
